NR & ReX Network Integration Project

This repository contains documentation for a simulated project
management exercise completed as part of the Introduction to
Project Management module at Victoria University.

Project Scope:
Integration of National Railways (NR) and Regional Express (ReX)
operations following a corporate acquisition.

Key Deliverables:
- Project Charter
- Project Plan
- Work Breakdown Structure (WBS)
- Gantt Chart
- Budget Report
- Execution Plan
- Project Closure Report
